QpsSingleBeam (Class)

Namespace: stdlibXtf.Packets

Implements: IPacket

Define a QPS single beam data packet.

Constructors

QpsSingleBeam() Initializes a new instance of the QpsSingleBeam class that has default zero values.
QpsSingleBeam(Byte[]) Initializes a new instance of the QpsSingleBeam class that contain the values extracted from the given byte array.
The size of array need to be at least of 64 bytes.
