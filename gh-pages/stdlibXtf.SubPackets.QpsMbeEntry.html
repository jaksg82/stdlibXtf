QpsMbeEntry (Class)

Namespace: stdlibXtf.SubPackets

Define the base data packet.

Constructors

QpsMbeEntry() Initializes a new instance of the QpsMbeEntry class that has default zero values.
QpsMbeEntry(Byte[]) Initializes a new instance of the QpsMbeEntry class that contain the values extracted from the given byte array.
The size of array need to be at least of 64 bytes.

Properties

Id Gets the beam ID.
Intensity Gets the signal strength.
Quality Gets the quality.
TwoWayTravelTime Gets the two way travel time, in seconds.
DeltaTime Gets the beam time offset. DeltaTime can be used for profilers to calculate the ping time per beam.
BeamAngle Gets the beam angle. BeamAngle convention: negative to port side, nadir beam 0 degrees, positive to starboard side.
TiltAngle Gets the tilt angle. TiltAngle convention: positive forward, negative backward (used for pitch steering).
