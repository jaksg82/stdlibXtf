RawCustomHeader (Class)

Namespace: stdlibXtf.Packets

Implements: IPacket

Define a custom raw data packet.

Constructors

RawCustomHeader() Initializes a new instance of the RawCustomHeader class that has default zero values.
RawCustomHeader(Byte[]) Initializes a new instance of the RawCustomHeader class that contain the values extracted from the given byte array.
The size of array need to be at least of 64 bytes.
