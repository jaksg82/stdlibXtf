CorrectionFlags (Enum)

Namespace: stdlibXtf.Enums

Specify if the data is stored raw or corrected.

Fields

None Not specific way of storage. This is a default value that can be considered equivalent to Range.
Range Sonar imagery stored as raw, range or slant-range.
Depth Sonar imagery stored as corrected, depth or ground-range.
