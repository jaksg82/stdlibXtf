stdlibXtf.Common (Namespace)

The stdlibXtf.Common namespace contains classes and interfaces that define generic types, in order to handle the reading of the XTF document files.

Classes

IndexEntry Define a index/value pair that is used to store and retrieve a packet location inside the XTF file.
PacketHeaderTypes Define a collection of the known packet header types.
SonarModelTypes Define a collection of the known sonar models.
StatCollection Define the container for the statistics of the document.
StatEntry Define the container for the statistics of a specific packet.
TypeEntry Define the entry for the collections of known sonar and packets.

Interfaces

IExports Interface that provide export methods.
IPacket Interface that provide the base properties of all the packages.
