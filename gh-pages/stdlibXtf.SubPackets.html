SubPackets (Namespace)

Namespace: stdlibXtf

The stdlibXtf.SubPackets namespace contains classes that represent the small packets of data associated to certain known packets.

Classes

BathySnippet0 Represent the header part of the bathy data attached to a PingHeader packet.
BathySnippet1 Represent the beam part of the bathy data attached to a PingHeader packet.
BeamXYZA Represent the processed beam of the bathy data attached to a PingHeader packet.
ChannelInfo Represent the channel information structure present, one time for each channel, in the document header.
PingChannelData Represent the channel data.
PingChannelHeader Represent the channel informations that can be unique from ping to ping.
QpsMbeEntry Represent the beam part of the bathy data attached to a multi beam PingHeader packet.
QpsMultiTxEntry Represent the beam part of the bathy data attached to a single beam PingHeader packet.
