ChannelTypes (Enum)

Namespace: stdlibXtf.Enums

Specify the type of the information stored in a channel.

Fields

Subbottom The data stored in this channel represent a subbottom sensor.
Port The data stored in this channel represent the port sensor of a side scan sonar.
Starboard The data stored in this channel represent the starboard sensor of a side scan sonar.
Bathymetry The data stored in this channel represent a multibeam echosounder sensor.
