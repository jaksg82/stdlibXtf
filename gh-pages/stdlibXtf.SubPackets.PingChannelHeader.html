PingChannelHeader (Class)

Namespace: stdlibXtf.SubPackets

Represent the channel informations that can be unique from ping to ping.

Constructors

PingChannelHeader() Initializes a new instance of the PingChannelHeader class that has default zero values.
PingChannelHeader(Byte[]) Initializes a new instance of the PingChannelHeader class that contain the values extracted from the given byte array.
The size of array need to be at least of 64 bytes.

Properties

ChannelNumber Gets the number of the channel. Typically: 0=port low frequency, 1=starboard low frequency, 2=port high frequency and 3=starboard high frequency.
DownSampleMethod Gets the method used for the downsample. 2=MAX, 4=RMS.
SlantRange Gets the slant range of the data in meters.
GroundRange Gets the ground range of the data in meters (SlantRange2 - Altitude2).
TimeDelay Gets the amount of time, in seconds, to the start of recorded data.
TimeDuration Gets the amount of time, in seconds, recorded. Typically SlantRange/750.
SecondsPerPing Gets the amount of time, in seconds, from ping to ping (SlantRange/750).
ProcessingFlags Gets the kind of processing used.
Frequency Gets the center transmit frequency for this channel.
InitialGainCode Gets the setting as transmitted by sonar.
GainCode Gets the setting as transmitted by sonar.
Bandwidth Gets the setting as transmitted by sonar.
ContactNumber Gets the contact number. Updated when contacts are saved in Target utility.
ContactClassification Gets the contact classification. Updated when contacts are saved in Target utility.
ContactSubNumber Gets the contact sub number. Updated when contacts are saved in Target utility.
ContactType Gets the contact type. Updated when contacts are saved in Target utility.
NumberSamples Gets the number of samples that will follow this packet. The number of bytes will be this value multiplied by the number of bytes per sample (ChannelInfo.BytesPerSample).
MillivoltScale Gets the maximum voltage, in milliVolts, represented by a full-scale value in the data.
If zero, then the value stored in ChannelInfo.VoltScale should be used instead.
ContactTimeOffTrack Gets the time off track to this contact, stored in milliseconds.
ContactCloseNumber Gets the number.
FixedVSOP Gets the fixed along-track size of each ping, in centimeters.
On multibeam systems this value need to be filled in to prevent Isis from calculating along-track ground coverage based on beam spread and spedd over ground.
Weight Gets the weighting factor passed by some sonars.
This value is mandatory for Edgetech digital sonars types 24, 35, 38 and Kongsberg SA type 48.
