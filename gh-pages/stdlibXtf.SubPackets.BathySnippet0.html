BathySnippet0 (Class)

Namespace: stdlibXtf.SubPackets

Represent the header part of the bathy data attached to a PingHeader packet.

Constructors

BathySnippet0() Initializes a new instance of the BathySnippet0 class that has default zero values.
BathySnippet0(Byte[]) Initializes a new instance of the BathySnippet0 class that contain the values extracted from the given byte array.
The size of array need to be at least of 74 bytes.
