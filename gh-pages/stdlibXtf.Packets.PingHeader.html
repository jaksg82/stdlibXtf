PingHeader (Class)

Namespace: stdlibXtf.Packets

Implements: IPacket

Define the base data packet.

Constructors

PingHeader() Initializes a new instance of the PingHeader class that has default zero values.
PingHeader(Byte[]) Initializes a new instance of the PingHeader class that contain the values extracted from the given byte array.
The size of array need to be at least of 256 bytes.
