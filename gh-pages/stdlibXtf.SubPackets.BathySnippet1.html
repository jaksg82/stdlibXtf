BathySnippet1 (Class)

Namespace: stdlibXtf.SubPackets

Represent the beam part of the bathy data attached to a PingHeader packet.

Constructors

BathySnippet1() Initializes a new instance of the BathySnippet1 class that has default zero values.
BathySnippet1(Byte[]) Initializes a new instance of the BathySnippet1 class that contain the values extracted from the given byte array.
The size of array need to be at least of 24 bytes.

Properties

HeaderSize Gets the size, in bytes, of this header packet.
DataSize Gets the size, in bytes, of the following data packets.
PingNumber Gets the sequential number of the ping.
Beam Gets the beam number, from 0 to total number minus 1.
SnipSamples Gets the snippet size, samples.
GainStart Gets the gain at start of snippet, 0.01 dB steps.
GainEnd Gets the gain at end of snippet, 0.01 dB steps.
FragOffset Gets the fragment offset, samples from ping.
FragSamples Gets the fragment size, samples.
