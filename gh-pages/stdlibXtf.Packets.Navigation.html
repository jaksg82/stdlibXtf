Navigation (Class)

Namespace: stdlibXtf.Packets

Implements: IPacket

Define the base data packet.

Constructors

Navigation() Initializes a new instance of the Navigation class that has default zero values.
Navigation(Byte[]) Initializes a new instance of the Navigation class that contain the values extracted from the given byte array.
The size of array need to be at least of 64 bytes.

Properties

MagicNumber Gets the number that identify the correct start of the packet.
HeaderType Gets the type of the packet header.
SubChannelNumber Gets the index number of which channels this packet are referred.
NumberChannelsToFollow Gets the number of channels that follow this packet.
NumberBytesThisRecord Total byte count for this packet, including the header and the data if available.
PacketTime Gets the packet recording time.
SourceEpoch Source Epoch Seconds since 1/1/1970, will be followed attitude data even to 64 bytes.
SourceEpochMicroseconds The Microsecond part of SourceEpoch. Range: 0 to 999999
TimestampValidity Time stamp validity:
0 = only receive time valid
1 = only source time valid
3 = both valid
RawCoordinateX Raw position from POSMV or other time stamped navigation source.
RawCoordinateY Raw position from POSMV or other time stamped navigation source.
RawAltitude Altitude, can hold real-time kinematics altitude
