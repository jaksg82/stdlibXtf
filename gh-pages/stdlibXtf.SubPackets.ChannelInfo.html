ChannelInfo (Class)

Namespace: stdlibXtf.SubPackets

Represent the channel information structure present, one time for each channel, in the document header.

Constructors

ChannelInfo() Initializes a new instance of the ChannelInfo class that has default zero values.
ChannelInfo(Byte[]) Initializes a new instance of the ChannelInfo class that contain the values extracted from the given byte array.
The size of array need to be at least of 128 bytes.

Properties

TypeOfChannel Gets the kind of channel (Subbottom, Port, Starboard or Bathymetry).
SubChannelNumber Gets the index for which this ChannelInfo is.
CorrectionFlag Gets the way the data are stored.
Polarity Gets the polarity of the data.
BytesPerSample Gets the quantity of bytes for each sample stored.
SamplesPerChannel Gets the number of samples per channel. This value will be overridden by the NumberSamples in PingChannelHeader subpacket.
ChannelName Gets the description of the channel.
VoltScale Gets how many volts are represented by a maximum sample value in the range -5.0 to +4.9998 volts.
Frequency Gets the center transmit frequency.
HorizontalBeamAngle Gets the horizontal beam angle, tipically 1 degree.
TiltAngle Gets the tilt angle, tipically 30 degrees.
BeamWidth Gets the 3 dB beam width, tipically 50 degrees.
OffsetX Gets the offset of MultiBeam system. Orientation of positive X is to starboard.
OffsetY Gets the offset of MultiBeam system. Orientation of positive Y is forward.
OffsetZ Gets the offset of MultiBeam system. Orientation of positive Z is down, just like depth.
OffsetYaw Gets the offset of MultiBeam system. Orientation of positive yaw is turn to right.
If the multibeam sensor is reverse mounted (facing backwards), then OffsetYaw will be around 180 degrees.
OffsetPitch Gets the offset of MultiBeam system. Orientation of positive pitch is nose up.
OffsetRoll Gets the offset of MultiBeam system. Orientation of positive roll is lean to starboard.
BeamsPerArray Gets the beams for each array. Valid only for forward looking sonar systems.
Latency Gets the latency in milliseconds.
