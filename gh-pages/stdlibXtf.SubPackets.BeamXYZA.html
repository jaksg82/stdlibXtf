BeamXYZA (Class)

Namespace: stdlibXtf.SubPackets

Represent the processed beam of the bathy data attached to a PingHeader packet.

Constructors

BeamXYZA() Initializes a new instance of the BeamXYZA class that has default zero values.
BeamXYZA(Byte[]) Initializes a new instance of the BeamXYZA class that contain the values extracted from the given byte array.
The size of array need to be at least of 31 bytes.
